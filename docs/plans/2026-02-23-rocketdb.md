# Local Cognition Device Management - RocksDB Data Lake Implementation

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 使用 RocksDB 实现本地认知设备管理系统的核心数据湖功能，支持设备状态存储、传感器时序数据、知识图谱和 HTTP API。

**Architecture:** 基于 Rust + RocksDB 实现高效键值存储，使用 Column Families 隔离不同类型数据，通过 Axum 提供 HTTP API 接口。

**Tech Stack:** Rust, RocksDB 0.22, Axum 0.7, Python Skills (Ollama), serde, chrono

### Task 5: Key 编码优化

**问题:** 当前使用字符串拼接 Key，效率较低。

**优化方案:** 参考原始设计，使用二进制 Key 编码：

**Files:**
- Modify: `src/cognition/data_lake.rs`

**Step 1: 添加 KeyBuilder**

```rust
use byteorder::{BigEndian, WriteBytesExt};
use std::io::Write;

pub struct KeyBuilder;

impl KeyBuilder {
    /// 构建二进制 Key: device_id + 0x00 + tag + 0x00 + timestamp (BigEndian)
    pub fn build(device_id: &str, tag: &str, timestamp: u64) -> Vec<u8> {
        let mut key = Vec::new();
        key.extend_from_slice(device_id.as_bytes());
        key.push(0x00);
        key.extend_from_slice(tag.as_bytes());
        key.push(0x00);
        key.write_u64::<BigEndian>(timestamp).unwrap();
        key
    }

    /// 构建前缀用于扫描
    pub fn build_prefix(device_id: &str, tag: &str) -> Vec<u8> {
        let mut prefix = Vec::new();
        prefix.extend_from_slice(device_id.as_bytes());
        prefix.push(0x00);
        prefix.extend_from_slice(tag.as_bytes());
        prefix.push(0x00);
        prefix
    }
}
```

**Step 2: 修改存储方法**

```rust
pub fn store_device_state(&self, state: &DeviceState) -> Result<()> {
    // 旧: let key = format!("device_{}_state_{}", state.device_id, state.timestamp);
    let key = KeyBuilder::build(&state.device_id, "state", state.timestamp);
    let value = serde_json::to_vec(state)?;
    self.db.put_cf(self.cf_device_state(), key, value)?;
    Ok(())
}
```

### Task 8: 批量写入优化

**问题:** ROS 高频数据单条写入效率低。

**Step 1: 添加批量写入方法**

```rust
use rocksdb::WriteBatch;

pub fn store_device_states_batch(&self, states: &[DeviceState]) -> Result<()> {
    let mut batch = WriteBatch::default();
    let cf = self.cf_device_state();

    for state in states {
        let key = KeyBuilder::build(&state.device_id, "state", state.timestamp);
        let value = serde_json::to_vec(state)?;
        batch.put_cf(cf, key, value);
    }

    self.db.write(batch)?;
    Ok(())
}
```
